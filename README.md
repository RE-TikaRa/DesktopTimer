# Desktop Timer Website

This branch (`gh-pages`) contains the static website for the Desktop Timer project.

## Structure

- `index.html`: The main landing page.
- `logo.svg`: Project logo.
- `favicon.ico`: Site favicon.
- `styles/main.css`: Custom site styles.
- `scripts/main.js`: Site interactions.
- `.nojekyll`: Bypasses Jekyll processing.

## Deployment

This branch is configured to be deployed via GitHub Pages.
